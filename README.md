# 🤖 AI Research Assistant (Gemini Chatbot)

A production-ready Generative AI application that acts as an intelligent research assistant, capable of answering questions related to Artificial Intelligence, Machine Learning, Deep Learning, and Generative AI using Large Language Models.

This project demonstrates an end-to-end AI pipeline for **real-time conversational intelligence, knowledge retrieval, and context-aware response generation**, making it suitable for educational, research, and productivity use cases.

---

## 🚀 Key Features

* 💬 Ask questions related to AI, ML, Deep Learning, and GenAI  
* 🧠 Context-aware intelligent responses using Gemini LLM  
* ⚡ Real-time conversational experience  
* 🎯 Clean and minimal UI built with Streamlit  
* 🔐 Secure API key handling using environment variables  
* 🌐 Ready for deployment on Streamlit Cloud  

---

## 🛠️ Tech Stack

* **Python**  
* **Streamlit** – Interactive UI  
* **Google Gemini API** – LLM-based response generation  
* **python-dotenv** – Secure environment variable management

## ⚙️ Installation & Setup

cd ai-research-assistant  
python -m venv venv  
venv\Scripts\activate   # Windows  
pip install -r requirements.txt  

---

## ⚙️ Usage & Details

### 🔑 Environment Configuration
Create a `.env` file in the root directory:  
GEMINI_API_KEY=your_api_key_here  

### ▶️ Run the Application
streamlit run app.py  

### 🧠 System Workflow
- Accept user research query from input box  
- Process input using Streamlit  
- Send query to Gemini API  
- Apply prompt-based LLM processing  
- Generate context-aware response  
- Display response in UI  

### 📂 Output
- Real-time generated responses displayed in UI  
- Interactive conversational output
